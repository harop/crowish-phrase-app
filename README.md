# LaravelによるCRUD開発入門ハンズオン

# 今回作成するアプリ
名言サービスを作成します。
ここでは名言の閲覧・登録などが可能です。

## ユースケース
- ユーザーは名言を登録・削除・更新・取得する事ができる

## CRUD処理とは？
システムに必要な「Create（生成）」「Read（読み取り）」「Update（更新）」「Delete（削除）」のことを指します。
